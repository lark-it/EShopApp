# EShopApp (E-commerce)

Учебный e-commerce проект на Android (Kotlin + Jetpack Compose).

На базе API: DummyJSON 

---

## Features
- Каталог категорий
- Список товаров по категориям
- Поиск товаров
- Экран информации о товаре
- Избранное (Room)
- Корзина (Room)

---

## Tech Stack
- Kotlin
- Jetpack Compose
- Coroutines + Flow / StateFlow
- Room
- Retrofit + OkHttp
- Hilt (DI)
- MVVM (architecture)
