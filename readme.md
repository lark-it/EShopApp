# EShopApp (E-commerce)

Учебный e-commerce проект на Android (Kotlin + Jetpack Compose).

На базе API: DummyJSON 

---

## Features
### Каталог и товары
- просмотр главного экрана с товарами
- просмотр списка категорий
- переход в товары конкретной категории
- поиск товаров
- экран подробной информации о товаре

### Работа с товарами
- добавление товаров в корзину
- изменение количества товаров в корзине
- удаление товаров из корзины
- добавление товаров в избранное
- хранение избранного локально

### Оформление заказа
- выбор адреса перед оформлением
- создание заказа на основе содержимого корзины
- автоматический расчёт количества товаров и итоговой суммы
- сохранение заказов локально
- просмотр истории заказов
- просмотр состава заказа

### Профиль
- экран профиля пользователя
- просмотр данных текущего пользователя
- список сохранённых адресов
- создание адреса
- редактирование адреса
- удаление адреса

### Авторизация
- вход через Google
- интеграция с Firebase Authentication
- скрытие экрана авторизации после успешного входа
- выход из аккаунта

---

## Tech Stack
- Kotlin
- Jetpack Compose
- Coroutines + Flow / StateFlow
- Room
- Retrofit + OkHttp
- Moshi
- Hilt (DI)
- MVVM (architecture)
- Coil
- Firebase Authentication
